# Candidate Lessons

This file accumulates observations the project owner thinks might generalize to the template. Low bar to add; no obligation to promote. Reviewed at milestones (end of phase, end of quarter, launch) or on demand.

The promotion path (see S/W Development Environment operating doctrine §6):

```
This file (local, low bar)
        ↓
  Promotion review in S/W Dev Env Claude project
        ↓
  Scrub protocol (never cross boundary with proprietary info)
        ↓
  Template LESSONS_LEARNED.md (upstream, versioned, public)
        ↓
  Optional constitution or scaffolding update
        ↓
  New template version
```

